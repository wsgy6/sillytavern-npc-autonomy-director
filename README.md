# NPC 自主性导演

这是一个可直接作为 GitHub / GitLab 第三方扩展仓库使用的 SillyTavern 扩展，用来让 NPC 在对话中持续保持自己的目标，并且让外在穿搭与行为表现以可控、渐进的方式演化。

## 本次修复

本版本修复了扩展详情面板“点开后自动缩回去”的问题。

根因是：

- 扩展自己重复绑定了折叠点击事件
- 同时 SillyTavern 原生也会处理 `.inline-drawer-toggle`
- 再叠加扩展内部频繁 `renderPanel()` 重绘，导致面板状态被反复覆盖

现在的修复方式是：

- 改为复用 SillyTavern 原生折叠逻辑
- 只监听原生折叠事件结果
- 用 `localStorage` 记住当前展开/收起状态
- 每次重绘后恢复展开状态

## 仓库用途

这个目录是为“从 Git 仓库直接安装”准备的独立版本。

仓库根目录直接包含：

- `manifest.json`
- `index.js`
- `style.css`
- `README.md`

## 安装方法

1. 在 GitHub 新建一个公开仓库。
2. 将本目录中的所有文件上传到仓库根目录。
3. 确认仓库根目录直接显示：
   - `manifest.json`
   - `index.js`
   - `style.css`
4. 在 SillyTavern / 云酒馆的扩展安装入口填写仓库地址。
5. 安装后刷新页面。

## 验证方式

1. 打开扩展详情面板。
2. 面板应保持展开，不会自行缩回。
3. 修改任意设置后，面板仍应保持当前展开状态。
4. 输入 `/npc-auto-status`，确认扩展功能仍正常。

## 回退方式

- 删除旧仓库内容，重新上传本目录内容即可
- 如需彻底停用，关闭扩展开关或删除扩展目录
